# GMSwitcher
This library is used like UISwitcher, it has some color and effect that user may need. Compatible with swift 2.0, iOS 9
GMSwitcher
Written by Nguyen Duc Hoang, September 2015.

##Installation

Open in Xcode7 SWIFT project.	

##Usage
@IBOutlet weak var switcher1:GMSwitcher!
switcher1.turnOn(1)
        

##Demo

You can open the SLPopupViewControllerDemo demo project in Xcode and run it on your iPhone as well as in the Simulator.
